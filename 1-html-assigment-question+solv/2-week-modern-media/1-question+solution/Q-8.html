What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure and layout of the content on a webpage.

What is Semantic HTML?

Semantic HTML refers to the use of HTML elements that convey meaning beyond just the visual presentation. It helps search engines and assistive technologies understand the content better and improves the overall accessibility and maintainability of a webpage.

Commonly Used Semantic Tags

  • <header>: Represents the header of a document or a section.
  • <nav>: Defines a section with navigation links.
  • <main>: Represents the main content of a document.
  • <article>: Represents a self-contained composition in a document.
  • <section>: Defines a standalone section of a document.
  • <aside>: Represents content that is tangentially related to the main content.
  • <footer>: Represents the footer of a document or a section.
