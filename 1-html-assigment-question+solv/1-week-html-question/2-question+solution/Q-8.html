HTML (Hypertext Markup Language)
HTML is the standard markup language for creating the structure and content of web pages.
CSS (Cascading Style Sheets)
CSS is used to describe the presentation and visual styling of HTML documents, allowing developers to control the layout, colors, fonts, and other aspects of the webpage.
JavaScript
JavaScript is a programming language that enables interactivity on web pages. It allows for dynamic content, user input validation, and enhances the user experience through client-side scripting.
React.js
React.js is a JavaScript library used for building user interfaces. It provides reusable UI components and efficiently updates and renders data-driven views, making it popular for creating interactive and responsive web applications.
Node.js
Node.js is a runtime environment that allows server-side execution of JavaScript. It provides a non-blocking, event-driven model, making it suitable for building scalable and efficient network applications, including web servers.
Express.js
Express.js is a web application framework for Node.js. It simplifies the process of building robust and scalable web applications by providing a set of features and utilities for routing, middleware, and handling HTTP requests and responses.
MySQL
MySQL is a popular open-source relational database management system. It is used to store and manage structured data, making it an essential component of many web applications.
MongoDB
MongoDB is a NoSQL database that uses a flexible document-based model. It is known for its scalability, high performance, and ease of use, making it suitable for handling large amounts of unstructured data.
REST API (Representational State Transfer)
REST is an architectural style used for designing networked applications. It defines a set of principles for building scalable and loosely coupled systems. REST APIs enable communication between different software components over the web.
GraphQL
GraphQL is a query language and runtime that provides an alternative to traditional REST APIs. It allows clients to specify the exact data they need, reducing the amount of data transferred over the network and enabling efficient data fetching.
Git
Git is a distributed version control system used for tracking changes in source code during software development. It enables collaboration, version management, and the ability to revert to previous states of a project.
